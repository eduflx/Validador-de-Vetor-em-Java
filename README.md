# Validador-de-Vetor-em-Java
Programa em Java que reliza uma validação se todos os elementos do vetor são distintos.
O Programa é baseado em 1 função dentro da classe principal: 

* Valida se todos são disitintos: Realiza uma validação para verificar se todos elementos são distintos.

O programa retorna os resultados dentro da classe principal, que pede a entrada para o usuário chama a funções e a executa.

Exercicio 5 da Lista de exercícios nº3 de Extrutura de Dados 1 do curso de ADS do IFTM Campus Patrocínio.
